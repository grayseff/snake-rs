## Snake.rs 

This is a very simple project designed to help learn rust. Mostly self-coded, using concepts like structs enums etc.

Some vibe-coding was necessary for the formatting of crossterm etc as I hadn't encountered this before. 

Should draw a snake and his snack in a 40x25 grid in a terminal.
The snake is controlled with the arrow keys, esc quits.
the walls are periodic, the food can't generate inside the snake. The snake dies if the head moves into himself
Your score is the number of snacks you eat before quitting or your snake dies.

Have fun!
